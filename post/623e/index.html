JavaScript 基础第二天

算术运算符:

  1. 数学运算符也叫算术运算符，主要包括加、减、乘、除、取余（求模）
  2. +：求和
  3. -：求差
  4. *：求积
  5. /：求商
  6. %：取模（取余数）
  7. 开发中经常作为某个数字是否被整除

算术运算符执行的优先级顺序:

  1. 同时使用多个运算符编写程序时，会按着某种顺序先后执行，我们称为优先级
  2. JavaScript中 优先级越高越先被执行，优先级相同时以书从左向右执
  3. 乘、除、取余优先级相同
  4. 加、减优先级相同
  5. 乘、除、取余优先级大于加、减
  6. 使用 () 可以提升优先级
  7. 总结： 先乘除后加减，有括号先算括号里面的
  8. 来判断某个数字是否能被整除

计算圆的面积:

赋值运算符:

  1. 赋值运算符：对变量进行赋值的运算符
  2. 已经学过的赋值运算符：= 将等号右边的值赋予给左边, 要求左边必须是一个容器
  3. 其他赋值运算符: +=, -=, *=, /=, %=
  4. 使用这些运算符可以在对变量赋值时进行快速操作
  5. let num = 10 ， num 加5 赋值写就是: num += 5

一元运算符:

  1. 众多的js运算符可以根据所需表达式的个数，分为一元运算符、二元运算符、三元运算符
  2. 经常用于计数来使用。 比如进行10次操作，用它来计算进行了多少次了
  3. 前置自增和后置自增单独使用没有区别
  4. 前置自增和后置自增如果参与运算就有区别
  5. 前置：先自增后运算
  6. 后置：先运算后自增

自增运算符的用法：

  1. 前置自增和后置自增独立使用时二者并没有差别
  2. 一般开发中我们都是独立使用, 后面 i++ 后置自增会使用相对较多

比较运算符:

  1. 作用：比较两个数据大小、是否相等
  2. 字符串比较，是比较的字符对应的ASCII码, 从左往右依次比较
  3. NaN不等于任何值，包括它本身
  4. 尽量不要比较小数，因为小数有精度问题
  5. 不同类型之间比较会发生隐式转换, 最终把数据隐式转换转成number类型再比较

比较运算符的使用:

  1. > ： 左边是否大于右边
  2. <： 左边是否小于右边
  3. =： 左边是否大于或等于右边
  4. <=： 左边是否小于或等于右边
  5. ==： 左右两边是否相等 ==
  6. ===： 左右两边是否类型和值都相等
  7. !==： 左右两边是否不全等
  8. 比较结果为boolean类型，即只会得到true或false

逻辑运算符:

  1. 逻辑运算符用来解决多重条件判断

逻辑运算符的使用:

符号名称读法特点口诀
&&逻辑与并且符号两边都为true 结果才为true一假则假
||逻辑或或者符号两边有一个 true就为true一真则真
!逻辑非取反true变false false变true真变假, 假变真

逻辑运算符里的短路:

  1. 短路：只存在于 && 和 || 中，当满足一定条件会让右边代码不执行
  2. &&: 左边为false就短路
  3. ||: 左边为true就短路
  4. 原因：通过左边能得到整个式子的结果，因此没必要再判断右边
  5. 运算结果：无论 && 还是 || ，运算结果都是最后被执行的表达式值，一般用在变量赋值
  1. 需求：用户输入一个，判断这个数能被4整除，但是不能被100整除

运算符优先级:

  1. 一元运算符里面的逻辑非优先级很
  2. 逻辑与比逻辑或优先级高

表达式和语句:

表达式是一组代码的集合，js解释器会将其计算出一个结果

表达式和语句的区别:
  1. js整句或命令，js语句是以分号结束（可以省略）, 比如: if语句 for 循环语句
  2. 表达式计算出一个值，但语句用来自行以使某件事发生(做什么事)
  3. 表达式: 3 + 4 , 语句: alert() 弹出对话框
  4. 其实某些情况，也可以把表达式理解为语句，因为它是在计算结果，也是做事

分支语句:

  1. 程序三大流程控制语句: 顺序, 分支, 循环
  2. 以前我们写的代码，写几句就从上往下执行几句，这种叫顺序结构
  3. 有的时候要根据条件选择执行代码，这种就叫分支结构
  4. 某段代码被重复执行，就叫循环结构
  5. 分支语句包含: if分支语句, 三元运算符, switch语句

  1. if语句有三种使用：单分支、双分支、多分支

3. Switch 语句:

  1. switch case语句一般用于等值判断,不适合于区间判断
  2. switch case一般需要配合break关键字使用 没有break会造成case穿透
  3. 找到跟小括号里数据全等的case值，并执行里面对应的代码
  4. 若没有全等 === 的则执行default里的代码
  5. 例：数据若跟值2全等，则执行代码2
  6. 全等判断, break：结束switch语句，防止穿透

简单计算器:

  1. 用户输入2个数字，然后输入 + - * / 任何一个，可以计算结果
  2. 用户输入不同算术运算符，可以去执行不同的运算 (switch)

4. 循环结构:

1. 循环结构:

  1. 作用：学习时可以帮助更好的理解代码运行，工作时可以更快找到bug
  2. 断点：在某句代码上加的标记就叫断点，当程序执行到这句有标记的代码时会暂停下来

2. While 循环:

  1. 循环：重复执行某段代码， 而 while : 在…. 期间

1. while 循环语法:

  1. 跟if语句很像，都要满足小括号里的条件为true才会进入执行代码
  2. while大括号里代码执行完毕后不会跳出，而是继续回到小括号里判断条件是否满足，若满足又执行大括号里的代码，然后再回到 小括号判断条件，直到括号内条件不满足，即跳出
  3. 循环的本质就是以某个变量为起始值，然后不断产生变化量，慢慢靠近终止条件的过程

2. 循环需要具备三要素：

  1. 变量起始值
  2. 终止条件（没有终止条件，循环会一直执行，造成死循环
  3. 变量变化量（用自增或者自减）

  1. continue：结束本次循环，继续下次循环
  2. break：跳出所在的循环
